使用国内镜像解决Python库安装慢

发表于 2018-02-17 | 分类于 Python

由于国内被墙或者其他原因，导致有些库安装很慢，可以使用国内镜像来解决这个问题，例如清华大学的pip源，它是官网pypi的镜像，每隔5分钟会跟官方库同步一次，地址为清华大学python库国内镜像，命令格式如下：

pip3 install -i https://pypi.tuna.tsinghua.edu.cn/simple pyqt5

上面命令通过安装PyQt5库来举例。
